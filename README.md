# Rustboy
[![Build Status](https://travis-ci.org/wez470/Rustboy.svg?branch=master)](https://travis-ci.org/wez470/Rustboy)

# Building / Running
1. Install SDL2 Dev library.
1. `cargo run --release run <ROM>`

# Controls
```
start   = enter
select  = tab
a       = k
b       = j
up      = w
left    = a
down    = s
right   = d
```
